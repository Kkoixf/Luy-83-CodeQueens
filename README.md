# Luy-83-CodeQueens
Este projeto apresenta a Luy-83, uma mão robótica inspirada na mobilidade humana, projetada para testes de manipulação e controle biométrico. Com múltiplos servos motores, a mão permite movimentos detalhados dos dedos e da palma, enquanto sensores embutidos capturam informações táteis e de pressão. Ideal para aplicações em robótica assistiva.

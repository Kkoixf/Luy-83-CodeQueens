Como Funciona a Conexão

O sistema Luy permite conectar dispositivos via Bluetooth e Wi-Fi para monitoramento e controle.

Para Administradores:

  • Acesse a aba "Conectar Wifi" para configurar a rede.
  • Use a aba "Controle" para gerenciar dispositivos conectados.
  • Na aba "Dados/Pacientes", visualize e edite informações dos pacientes.

Conexão Bluetooth:

Certifique-se de que o dispositivo Bluetooth esteja ativado no aparelho. O sistema detectará automaticamente dispositivos compatíveis.

Conexão Wi-Fi:

Configure a rede Wi-Fi através da aba dedicada. Insira as credenciais da rede para estabelecer a conexão.
